Go to "preferences..." and add these git repository URLs:
http://arduino.esp8266.com/stable/package_esp8266com_index.json
https://github.com/earlephilhower/arduino-pico/releases/download/global/package_rp2040_index.json

Install WiFi library:
Arduino Uno WiFi Dev Ed Library

Add ESP32 Boards:
Esp32 by Espressif Systems

Select Board:
ESP32-WROVER or ESP32-WROOM. Both work for these boards despite being WROOM

After pressing compile, you will eventually "see connecting....", at this point, press and hold the right buttong
until the upload process begins
